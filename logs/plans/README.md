# Plan Logs

Thư mục này lưu log triển khai theo plan. Mỗi log nên liên kết tới plan tương ứng trong `plans/` và doc tương ứng trong `docs/` nếu có.

Nội dung nên ngắn:

- Mục tiêu.
- Phase đã làm.
- File đã sửa.
- Verify đã chạy.
- Blocker/rủi ro còn lại.

Không copy output dài hoặc dữ liệu nhạy cảm vào log.
