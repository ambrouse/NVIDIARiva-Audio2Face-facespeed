# Benchmark Runs

Thư mục này lưu kết quả chạy benchmark dạng máy đọc được và báo cáo case chi tiết.

| Path | Nội dung |
| --- | --- |
| `matrix/` | Các batch benchmark theo số file, số câu và concurrency. |

Reviewer nên bắt đầu từ `tests/benchmarks/README.md`, sau đó mở `summary.json`, `results.json` hoặc `case-log.md` trong run cụ thể nếu cần kiểm chứng từng case.
