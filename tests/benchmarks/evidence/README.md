# Benchmark Evidence

Thư mục này lưu evidence có thể review trên GitHub cho các lần kiểm chứng RAG Voice.

| Folder | Trạng thái | Nội dung |
| --- | --- | --- |
| `rag-voice-2026-05-25-blocked/` | blocked | Evidence khi provider chính chưa online. |
| `rag-voice-2026-05-25-smoke/` | smoke | Evidence trong quá trình sửa pipeline. |
| `rag-voice-2026-05-25-final/` | pass | Evidence cuối: frontend, audio input/output, ASR, animation và browser report. |

Ảnh/audio/video ở đây chỉ có giá trị khi đi kèm report giải thích input, output và kết quả pass/fail. Không đưa secret, token hoặc dữ liệu production vào evidence.
