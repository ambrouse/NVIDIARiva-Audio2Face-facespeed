# Benchmark Corpus

Thư mục này lưu dữ liệu PDF dùng cho benchmark RAG Voice.

| Path | Nội dung |
| --- | --- |
| `arxiv/` | PDF arXiv thật, metadata download và kết quả ingest. |

Corpus này là dữ liệu kiểm thử, không phải tài liệu vận hành. Khi thay đổi corpus, cập nhật report benchmark để ghi rõ số PDF, số chunk và ngày chạy.
