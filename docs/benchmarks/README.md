# Benchmark Docs

Thư mục này lưu tài liệu benchmark dạng dễ đọc cho reviewer và người vận hành. Nội dung ở đây nên trỏ về artifact thật trong `tests/benchmarks/` hoặc `logs/benchmarks/`, không tự tạo số liệu mới nếu chưa có evidence.

Quy ước:

- Báo cáo benchmark tổng quan đặt theo ngày hoặc tên chiến dịch.
- `LATEST.md` trỏ tới báo cáo mới nhất nếu có.
- Khi benchmark thay đổi gate pass/fail, cập nhật cả `tests/benchmarks/README.md`.
