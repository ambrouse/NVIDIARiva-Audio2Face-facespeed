# Phase Reports

Thư mục này lưu báo cáo phase lịch sử của dự án NVIDIA/Riva/Audio2Face. Các file có thể không phản ánh trạng thái runtime mới nhất, nhưng giữ lại để tra cứu quyết định kỹ thuật và evidence theo từng giai đoạn.

Khi thêm report mới:

- Ghi ngày hoặc phase rõ ràng.
- Nêu mục tiêu, thay đổi, evidence, verify và rủi ro còn lại.
- Không đưa secret, token hoặc log runtime dài vào report.
