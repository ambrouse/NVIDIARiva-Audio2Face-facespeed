# Documentation Assets

Thư mục này lưu ảnh, GIF hoặc media dùng trong README và docs.

Quy tắc:

- Chỉ lưu asset thật sự được link từ tài liệu.
- Tên file nên mô tả nội dung.
- Không lưu secret, ảnh có token, hoặc ảnh có dữ liệu nhạy cảm.

## Asset Hiện Có

| File | Nội dung |
| --- | --- |
| `facespeed-readme-banner.png` | Banner README mới, chụp từ app thật qua `http://127.0.0.1:6300/`, mở Sources popup với 100 PDF đã index. |
| `voice-rag-avatar-demo.gif` | GIF demo cũ từ browser app. |
| `facespeed-demo.gif` | GIF demo lịch sử. |
