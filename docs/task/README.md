# Task Docs

Thư mục này lưu tài liệu theo từng task thay đổi repo. Mỗi file nên có ngày, phạm vi, kết quả, cách verify và liên kết ngược tới plan/log nếu task đủ lớn.

Quy ước tên file:

```text
task-{task-name}-YYYYMMDD-v1.md
```

Nội dung ở đây khác `docs/plan/`: `docs/task/` ghi lại kết quả thực tế và quyết định sau khi làm, còn `docs/plan/` lưu kế hoạch hoặc thiết kế trước/sát lúc triển khai.

| --- | --- |
