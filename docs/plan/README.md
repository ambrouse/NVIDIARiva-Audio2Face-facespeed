# Planning Docs

Thư mục này lưu tài liệu nền cho kế hoạch triển khai: phân tích thiết kế, phạm vi phase, quyết định kỹ thuật và ghi chú handoff.

Khác biệt với `plans/`:

- `plans/` là checklist/phase tracker có status.
- `docs/plan/` là tài liệu giải thích bối cảnh, lựa chọn và phạm vi để người khác hiểu vì sao plan tồn tại.

Mỗi tài liệu mới nên có ngày hoặc version nếu mô tả một task cụ thể.
